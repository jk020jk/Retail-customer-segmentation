# Customer Segmentation using RFM Analysis

## Problem
Identify high-value customers and at-risk segments for targeted marketing.

## Dataset
UCI Online Retail Dataset — 137,839 transactions, 2093 unique customers

## Approach
- Data cleaning using Python
- SQL (SQLite) for RFM metric calculation
- Customer segmentation into 5 groups

## Key Finding
Champions (25% of customers) generate 63% of total revenue.
800 lost customers represent significant recovery opportunity.

## Tools
Python, Pandas, SQLite, Matplotlib

## Business Impact
Targeted retention of At Risk segment (239 customers) can prevent revenue loss.
